## セマンティックHTMLとは

セマンティックHTMLとは、コンテンツの意味や構造を正しく伝えるHTMLタグを使用する手法のことです。`<div>` や `<span>` のような汎用タグに頼るのではなく、コンテンツの役割に合ったタグを選ぶことで、ブラウザや支援技術がページの構造をより正確に理解できるようになります。

## なぜセマンティックHTMLが重要なのか

### アクセシビリティの向上

スクリーンリーダーなどの支援技術は、HTMLタグの意味を基にコンテンツをユーザーに伝えます。正しいタグを使うことで、視覚に障害のあるユーザーにも適切な情報が届きます。

```html
<!-- 悪い例 -->
<div class="nav">
  <div class="nav-item">ホーム</div>
  <div class="nav-item">概要</div>
</div>

<!-- 良い例 -->
<nav>
  <ul>
    <li><a href="/">ホーム</a></li>
    <li><a href="/about">概要</a></li>
  </ul>
</nav>
```

### SEOへの効果

検索エンジンはHTMLの構造を解析してページの内容を理解します。セマンティックなマークアップを使うことで、検索エンジンがコンテンツの意味を正しく把握し、適切にインデックスできます。

## 主要なセマンティックタグ

| タグ | 用途 |
|------|------|
| `<header>` | ページやセクションのヘッダー |
| `<nav>` | ナビゲーションリンク |
| `<main>` | メインコンテンツ |
| `<article>` | 独立したコンテンツ |
| `<section>` | テーマごとのグループ |
| `<aside>` | 補足情報 |
| `<footer>` | フッター |

## 実践的なページ構造

```html
<!DOCTYPE html>
<html lang="ja">
<head>
  <meta charset="UTF-8">
  <title>セマンティックHTML</title>
</head>
<body>
  <header>
    <h1>サイトタイトル</h1>
    <nav>
      <ul>
        <li><a href="/">ホーム</a></li>
        <li><a href="/blog">ブログ</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <article>
      <h2>記事タイトル</h2>
      <p>記事本文...</p>
    </article>
  </main>

  <footer>
    <p>&copy; 2022 My Site</p>
  </footer>
</body>
</html>
```

## まとめ

セマンティックHTMLを正しく使うことは、アクセシビリティ・SEO・保守性すべてに良い影響を与えます。`<div>` の乱用を避け、コンテンツの意味に沿ったタグを選択する習慣をつけましょう。
